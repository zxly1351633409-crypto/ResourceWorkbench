# Changelog

## 0.3.1 - 2026-07-11

- 正式移动恢复完整计划预览，并要求精确输入 `MOVE`；取消、关闭或输错不会移动。
- 压缩包卡片重新分析改为明确的“不解压外层分析”，删除误导性的自动解压承诺。
- 资源树目录枚举、子目录探测和刷新移到后台，增加 generation 防旧结果覆盖、400 项分页、继续加载与失败重试。
- SpeedTree 优先复用真实 `preview.png` / `<工程名>_preview.png`，排除材质通道贴图；无图时使用明确非渲染占位封面。
- 运行数据维护扩展到 staging、资源索引和明确终态历史；后台执行，保留活动任务、可撤销记录和人工元数据。
- 修复窗口关闭时后台 Qt 任务的生命周期，避免退出阶段 fast-fail。
- Windows EXE FileVersion/ProductVersion 更新为 0.3.1。

## 0.3.0 - 2026-07-11

- 统一本地路径、压缩包和网页链接入口；分析与取消拆为独立按钮。
- 后台分析显示进度、取消入口和非阻塞状态提示；Windows 压缩包子进程不再弹黑框。
- 默认不读取或拆分压缩包内部贴图目录，首层目录数量与卡片数量执行硬校验。
- 分类结合批次、标题、文件样例、真实资源树和已确认移动历史，输出更深层现有目录建议。
- 修复资源库叶目录图片/视频预览、实际视频类型、深层目录浏览、自动刷新和侧栏新建文件夹。
- 增加语义主题颜色、中文设置/颜色选择器、缓存与移动日志保留策略。
- 开发、个人正式和干净分发 profile 隔离；ZIP 构建增加隐私/凭证/运行数据安全门。
- Windows EXE 写入 FileVersion/ProductVersion 0.3.0。

## 0.2.x

- 建立扫描、卡片墙、预览、翻译/审阅、目标建议、移动/撤销和网页资源基础流程。
